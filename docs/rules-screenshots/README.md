# Rules Screenshots

Store the official Musicathon 2026 rules screenshots here as evidence for
[`../CONTEST_RULES.md`](../CONTEST_RULES.md).

Before committing screenshots:

- Redact emails, usernames, tokens, API keys, private URLs, and browser profile details.
- Keep only the rules content and organizer branding needed to verify the transcription.
- Prefer ordered filenames such as `01-overview.png`, `02-eligibility.png`, etc.
- Do not add screenshots from private chats unless the organizer allows public sharing.

The text transcription remains in `CONTEST_RULES.md`; this folder is only for
source evidence.
